<Default
  title="Slide Default"
  subtitle="Titre standard plus contenu MDX libre dans le slot"
  highlight="Default"
>
  Le composant `Default` reprend le SlideTitle standard et expose un slot MDX pour insérer du contenu arbitraire dessous.

  Utile pour les slides qui ne tombent dans aucun layout spécialisé : prose longue, mélange de paragraphes et de listes, schéma SVG inline, embed YouTube ponctuel.

  Reste sobre. Si la même mise en forme revient deux fois, créer un composant dédié plutôt que d'abuser de `Default`.
</Default>

<Custom align="center">
  Composant escape-hatch sans SlideTitle ni structure imposée. Vous écrivez votre HTML ou Markdown directement dans le slot, avec le fond crème de la charte mais aucun cadre éditorial.

  À réserver aux cas où aucun layout n'a de sens et où la réutilisation n'est pas anticipée. Si vous y revenez deux fois, c'est probablement le signal pour créer un layout dédié dans `src/components/slides/`.
</Custom>
